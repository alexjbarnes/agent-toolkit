# Security Policy

## Reporting a Vulnerability

<!-- TODO: customise -- replace with your repo URL -->

If you discover a security vulnerability, report it privately through GitHub's security advisory feature:

1. Go to the [Security Advisories](https://github.com/yourorg/your-project/security/advisories) page
2. Click "Report a vulnerability"
3. Fill in the details

Do not open a public issue for security vulnerabilities.

## Response

Fixes for confirmed vulnerabilities will be released as patch versions.
